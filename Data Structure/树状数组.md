# <font color="green">树状数组</font>

虽然广泛的被叫做树状数组，但其实从它的英文名Binary Indexed TREE（二进制下表树）才完整表达了其原理。

### 适用方向：

能够以$O(\log_2N)$的复杂度完成单点修改和区间求和。
普通树状数组维护的信息及运算要满足：

$$
\begin{cases}
(x \circ y) \circ z = x \circ (y \circ z)\\
\exist f,y=f(x\circ y,x)\\
\end{cases}
$$

### 思路

寻常的数组支持以 $O(1)$ 进行单点修改，但区间求和复杂度为 $O(N)$ ，而利用前缀和维护区间和后，区间求和复杂度变为 $O(1)$ ，但单点修改却降至 $O(N)$ 。
我们需要一种折中的方案：
树状数组预处理了不完整的区间和，从而保证单点修改时只需修改 $\log_2N$ 数量级的区间和，区间求和时利用 $\log_2N$ 数量级的区间组合出答案，将整体时间复杂度优化至 $O(\log_2N)$ 。

### 具体实现

> <font face = "仿宋">Fenwick当年选择了二进制作为保留区间的选择标准，所以这里也用</font>

<font color= "brown">我们约定下面提到的数组均从下标1开始</font><br>
定义：

> 原数组为 $a[i]$ <br>
> $lowbit(i)$ 表示$i$在二进制下最低为1位对应数值

引入数组 $t[i]$ 满足
$$t[i]=\Sigma_{k=i-lowbit(i)+1}^i,$$

##### Degree 1.区间求和

$$\Sigma_{i=l}^ra[i]=\Sigma_{i=1}^ra[i]-\Sigma_{i=1}^la[i]$$
这样我们把区间求和转化成了前缀和问题
设数列 ${P_i}$ 满足 $k=2^{P_1}+2^{P_2}+\cdots+2^{P_n}$ 且$P_1<P_2<\cdots<P_n$ ，有
$$\Sigma_{i=1}^ka[i]=\Sigma_{i=1}^nt[k-\Sigma_{j=1}^i2^{p_j}]$$
显然，这一步是 $O(\log_2N)$ 的

##### Degree 2.单点修改

当我们修改 $a[i]$ 时，需要修改所有包含 $a[i]$ 的 $t[\Delta]$
不难发现 $t[i]$ 就是我们要修改的目标之一并且是其中下标值最小的一个
如何利用 $t[i]$ 寻找下一个要修改的区间呢？
我们先设下一个是 $t[k]$ ，那么 $k$ 满足：

$$
\begin{cases}
m-lowbit(m)\leq i\quad(1)\\
m>i\quad(2)\\
k = \inf(m)\quad(3)\\
\end{cases}
$$

尝试去解这个方程：
注意到若 $lowbit(m)<lowbit(i)$ ，那么 $m-lowbit(m)\geq i$ ,方程组无解。
故， $lowbit(i)\leq lowbit(m)$ 。
又因为变形可得
$$m\leq i+lowbit(m)\Rightarrow m+lowbit(i)\leq i+lowbit(i)+lowbit(m)$$
显然 $m\leq i+lowbit(i)$上式成立
若 $m<i+lowbit(i)$ ，由于$lowbit(i)\leq lowbit(m)$ ，只有$m=i$ ，与$(2)$矛盾舍去。
再看 $m=i+lowbit(i)$ 符合 $(1)(2)$ ，且为最小，故
$$k=i+lowbit(i)$$
显然这一性质可以对 $i+lowbit(i)$ 再次使用，由此我们得出全部需要修改的区间。

##### Degree 3.lowbit()求法

利用补码有关知识，我们可以得到$lowbit(i)=(-i)\& i$。

```
#include <cstdio>
#include <cstring>
#define MAXN 50005
#define lowbit(x) ((x) & (-x))
int tree[MAXN];
inline void update(int i, int x)
{
    for (int pos = i; pos < MAXN; pos += lowbit(pos))
        tree[pos] += x;
}
inline int query(int n)
{
    int ans = 0;
    for (int pos = n; pos; pos -= lowbit(pos))
        ans += tree[pos];
    return ans;
}
inline int query(int a, int b)
{
    return query(b) - query(a - 1);
}
int main()
{
    int cases;
    scanf("%d", &cases);
    for (int I = 1; I <= cases; ++I)
    {
        memset(tree, 0, sizeof(tree));
        int n, x, a, b;
        char opr[10];
        printf("Case %d:\n", I);
        scanf("%d", &n);
        for (int i = 1; i <= n; ++i)
        {
            scanf("%d", &x);
            update(i, x);
        }
        while (scanf("%s", opr), opr[0] != 'E')
        {
            switch (opr[0])
            {
            case 'A':
                scanf("%d%d", &a, &b);
                update(a, b);
                break;
            case 'S':
                scanf("%d%d", &a, &b);
                update(a, -b);
                break;
            case 'Q':
                scanf("%d%d", &a, &b);
                printf("%d\n", query(a, b));
            }
        }
    }
    return 0;
}
```
