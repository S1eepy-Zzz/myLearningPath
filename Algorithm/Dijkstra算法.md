# Dijkstra算法

dijkstra算法用于求解单源非负权最短路，时间复杂度为$O(N^2)$。

### 流程

> - 先给出具有 $1$到 $n$号节点的带非负权图，以邻接矩阵 $vp[i][j]$存储。
> - 记起点为 $st$, 用 $dis[i]$实时记录计算出的从$st$到$i$的当前最短路长度，显然，$dis[st]=0， Vis=\empty$。
> - 选择 $\min (\{dis[i]|i=1,2,\cdots,n \& i\notin Vis\})$, 记下标为 $mid$
>   - Vis中增加mid
>   - 令 $y\in \{i|vp[mid][i] != -1\}$，$dis[y]=\min (dis[y],vp[mid][y]+dis[mid])$
>   - 重复至 $Vis={i|i=1,2,\cdots ,n}$

### 正确性验证

**直观来讲**
当所有边长都是非负数的时候，全局最小值不可能再被其他节点更新，我们不断选择全局最小值进行拓展，最终可以得到起点到每个节点的最短路径的长度。
**严格来讲**
第一次加入的结点 $st$显然是最短路径。
假设第 $1,2,\cdots ,k-1$次加入的节点 $a_1,a_2,\cdots,a_{k-1}$为最短路。
现证明根据上述过程加入的第 $k$个结点为最短路径：
假设加入的$a_k$不是最短路径
显然，此时的最短路径至少存在一个节点不属于 $\{a_i\}$
设 $a_u$为**第一个**不属于 $\{a_i\}$的结点
由上述过程，此时应该加入 $a_u$而非 $a_k$，矛盾，得证。
